# Sound Assets

This folder contains audio files for the age-appropriate feedback system.

## Required Sound Files

### Celebration Sounds
- `celebration.mp3` - General celebration sound for all age groups
- `badge_unlock.mp3` - Sound when a badge is unlocked
- `star_collect.mp3` - Sound when stars are earned (lower-primary)
- `progress_fill.mp3` - Sound when progress bar fills (upper-primary)

### Character Sounds (Pre-Primary Ages 3-5)
- `lion_roar.mp3` - Strong voice feedback
- `mouse_squeak.mp3` - Soft voice feedback
- `elephant_trumpet.mp3` - Just-right voice feedback

### UI Sounds
- `click.mp3` - Button click sound
- `success.mp3` - General success sound

## File Requirements

- Format: MP3 (recommended) or WAV
- Sample rate: 44.1kHz
- Bit depth: 16-bit
- Duration: 1-3 seconds max
- File size: Keep under 100KB each

## Recommended Free Sound Resources

1. Freesound.org - CC licensed sounds
2. Mixkit.co - Free sound effects
3. Pixabay.com/sound-effects - Royalty-free sounds

## Notes

- All sounds should be child-friendly and pleasant
- Character sounds should be fun and recognizable
- Celebration sounds should be exciting but not too loud
- Keep sounds short to avoid annoying repetition
